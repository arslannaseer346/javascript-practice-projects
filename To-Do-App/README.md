📝 #To-Do App

A simple and interactive To-Do List app built with HTML, CSS, and JavaScript.

🚀 #Features

Add new tasks

Edit tasks directly from the list

Delete tasks

Prevent adding empty tasks

Updated tasks stay in the same place

Clean, centered, and responsive design

🛠 #Technologies

HTML – structure

CSS – styling and layout (flexbox)

JavaScript – dynamic behavior

💡 #How It Works

Type a task in the input field

Click Add → task is added

Click Edit → task text appears in input for modification

Click Delete → task is removed

⚡ #Future Improvements

Mark tasks as completed

Save tasks using local storage

Dark mode and animations
